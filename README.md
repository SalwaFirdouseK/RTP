# RTP
This project is a simple Java-based web server that allows users to scrape book data (title and price) from a given URL using JSoup.It provides a minimal web interface where users can input a URL, and the server returns scraped results in a table format.
Features:
Lightweight HTTP server using com.sun.net.httpserver
Web scraping using JSoup
Simple HTML UI with form input
Displays:
Book Title
Price
Handles errors gracefully
UTF-8 encoding support
